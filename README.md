# Note App

A basic note application built using React.js, CSS and JSON Server.

## Why this exists

This project was created to practice and strengthen React.js skills.

## What I'm practicing

- React fundamentals
- React Router fundamentals
- Component-based architecture
- Fetching and managing data from JSON Server

## How to start

1. Install dependencies:
```bash
npm install
```

2. Start JSON server:
```bash
npx json-server --watch data/db.json --port 4000
```

3. Start the development server:
```bash
npm run dev
```
